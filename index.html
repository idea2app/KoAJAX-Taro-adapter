    KoAJAX Taro adapter

    A KoAJAX adapter for mini-app HTTP requests.

    Usage

    Installation

    npm i koajax koajax-taro-adapter

    Initialization

    import { HTTPClient } from 'koajax';
    import { request } from 'koajax-taro-adapter';

    export const client = new HTTPClient({ baseRequest: request });
